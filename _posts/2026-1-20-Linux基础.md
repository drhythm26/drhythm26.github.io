## 命令提示符
```bash
[root@web200 ~]#
# root 当前登录用户 root 系统为管理员
# @为分隔符
# web200 为当前主机名
# ~所在位置显示代表当前所在目录，~ 指代当前登录系统的用户的家目录
# #井号表示为系统管理员 $美元符表示为普通用户
# eg
[root@web200 ~]# pwd
/root
[root@web200 ~]# cd /etc
[root@web200 etc]#
/etc
```
## linux文件系统
linux文件系统采用层次化的树状目录结构，一切文件目录都从根开始，没有Windows上的C盘D盘这样的盘符的概念
![](/assets/img/sample/linux基础/linux-directory-tree-diagram2026-01-20.png){: width="700" height="400" }
_linux-directory-diagram_
1. 绝对路径：以`/`开始的路径称为绝对路径
2. 相对路径：不以`/`开始，以当前目录为起点的路径称为相对路径
3. `.`表示当前目录
4. `..`表示上一级目录
5. `~`表示用户家目录

##  Shell命令
### pwd
pwd -> print working dirctory 打印当前工作目录
```bash
[root@web200 network-scripts]# pwd
/etc/sysconfig/network-scripts

[root@web200 system-connections]# pwd
/etc/NetworkManager/system-connections
```

### cd
cd -> change directory 切换目录
```bash
[root@web200 ~]# cd /usr/bin/ # 绝对路径 切换到指定目录下
[root@web200 bin]# pwd
/usr/bin

[root@web200 ~]# cd ..  # 相对路径 且换到上一级目录
[root@web200 /]# pwd
/

[root@web200 bin]# cd ~ # cd ~ 可以切换到用户家目录
[root@web200 ~]# pwd
/root

[root@web200 bin]# cd # 等同于 cd ~
[root@web200 ~]# pwd
/root

[root@web200 ~]# cd - # cd - 可以切换到上一次所在目录
/usr/bin
[root@web200 bin]# cd -
/root
```
