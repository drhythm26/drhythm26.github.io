## 主机密钥验证失败
```bash
## 报错信息
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
@    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
IT IS POSSIBLE THAT SOMEONE IS DOING SOMETHING NASTY!
Someone could be eavesdropping on you right now (man-in-the-middle attack)!
It is also possible that a host key has just been changed.
The fingerprint for the ED25519 key sent by the remote host is
SHA256:2KQMeZxBKX5ZzQxNtSpQ4l6LZPajRClMfnp4Fhm/PdA.
Please contact your system administrator.
Add correct host key in /home/lan/.ssh/known_hosts to get rid of this message.
Offending ECDSA key in /home/lan/.ssh/known_hosts:13
Host key for 10.0.0.10 has changed and you have requested strict checking.
Host key verification failed.

## 解决
# ip
ssh-keygen -R 10.0.0.10 
# ip + port
ssh-keygen -R "[10.0.0.10]:22"
# domain
ssh-keygen -R example.com
```
